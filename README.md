# 🚗 Valetrix – AI-Powered Valet Parking Assistant

Valetrix is a smart valet parking management system designed to streamline the parking experience for customers, drivers, security personnel, and administrators. The application enables users to book valet services, track vehicle status in real time, and efficiently manage parking operations through dedicated dashboards.

---

## 📖 Overview

Finding parking spaces and managing vehicle handovers can be time-consuming and inefficient. 
Valetrix provides a digital solution that automates the valet parking workflow by allowing customers to request valet services, 
assigning drivers, tracking vehicle movements, and ensuring secure vehicle handovers.

---

## ✨ Features

### 👤 Customer
- User Registration and Login
- Book valet parking
- Schedule vehicle pickup and return
- View booking history
- Track valet driver location
- Receive parking status updates
- Secure vehicle return verification

### 🚗 Driver
- Accept assigned parking requests
- View assigned bookings
- Navigate to customer location using Google Maps
- Update vehicle status
- Mark vehicle as parked or returned

### 🛡️ Security
- Verify customer identity before vehicle handover
- Confirm vehicle pickup and return
- Monitor parking activities
- Enhance parking security through verification

### 👨‍💼 Admin
- Manage customers and drivers
- Assign drivers to bookings
- Manage parking slots
- Monitor parking occupancy
- View booking reports
- Manage system users
